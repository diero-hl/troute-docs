# Roadmap

  No fixed dates. Each phase ships when it is ready.

  ## Phase 1: Foundation (Complete)

  - $TROUTE token deployed on Tempo Chain
  - Presale contract audited and open to the public
  - Twitter and Telegram communities launched
  - GitBook documentation published

  ## Phase 2: Developer Infrastructure (Complete)

  - Routing engine v1 deployed on Tempo Chain with full MPP integration
  - REST API live at api.trouteprotocol.xyz
  - TROUTE Explorer live at explorer.trouteprotocol.xyz
  - Token Launchpad API live on mainnet
  - Developer docs published: payment routing, billing, offramp, launchpad, integration
  - OpenAI function schema live so any agent can call TROUTE as a GPT Action

  ## Phase 3: Liquidity

  Presale closes. TGE happens. The USDC.e raised goes directly into seeding the LP on Tempo Native DEX. $TROUTE starts trading. Liquidity locked.

  - Presale closes and funds finalized
  - TGE executed, $TROUTE distributed to presale buyers
  - Liquidity pool seeded on Tempo Native DEX
  - $TROUTE trading live
  - Liquidity locked

  ## Phase 4: Launchpad

  Anyone launches a token on Tempo Chain. Humans connect a wallet and fill a form. AI agents call POST /api/launch. Either way the token is live in seconds on a bonding curve. $3 flat fee to launch. 1% on every trade, split 80% to the creator, 10% to stakers, 10% to treasury. At $20,000 market cap the token graduates automatically to Enshrined DEX with permanent liquidity.

  - Human launch: connect wallet, fill form, token live on Tempo Chain
  - AI agent launch: POST /api/launch, no wallet needed
  - $3 pathUSD flat launch fee
  - 1% trade fee split: 80% creator, 10% stakers, 10% treasury
  - Bonding curve graduates at $20,000 market cap
  - Auto-liquidity on Enshrined DEX at graduation

  ## Phase 5: Protocol Growth

  - Staking contract live, $TROUTE holders earn from all protocol fees
  - TROUTE SDK released for Python and TypeScript
  - First live AI agent integrations
  - 10+ active AI agents routing payments via TROUTE

  ## Phase 6: Scale

  - Governance live, stakers vote on fees and upgrades
  - Referral system live, earn from fees you bring in
  - $TROUTE fee discounts for holders
