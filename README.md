# Introduction

TROUTE is an autonomous payment routing protocol native to Tempo Chain.

It was built to solve a problem that nobody talks about but everyone building with AI will eventually hit: AI agents cannot move money on their own.

Not because the technology does not exist. But because no one built the infrastructure for it until now.

TROUTE is that infrastructure.

At its core, TROUTE is a routing layer that sits on top of Tempo Chain's Machine Payments Protocol (MPP). It handles the entire payment lifecycle for AI agents, from instruction to routing to settlement, without any human involvement at any step.

This documentation is for anyone who wants to understand how TROUTE works, why it was built on Tempo Chain, what $TROUTE does inside the protocol, and how to get involved early.
