# How to Get USDC.e

  To buy $TROUTE in the presale, you need USDC.e on Tempo Chain. Here is how to get it.

  ## What is USDC.e?

  USDC.e is USDC bridged to Tempo Chain. It is the only token accepted in the TROUTE presale and the settlement currency across the Tempo Chain ecosystem.

  Contract address on Tempo Chain: `0x20c000000000000000000000b9537d11c60e8b50`

  ## Step 1: Get USDC

  Buy USDC on Coinbase, Binance, Kraken, or any major exchange. Withdraw to your EVM wallet on Ethereum, Arbitrum, or Base.

  ## Step 2: Bridge to Tempo Chain

  **Tempo Bridge (Recommended)**

  Go to bridge.tempo.xyz. Select USDC, select Tempo Chain as destination, confirm. USDC arrives as USDC.e in 2 to 10 minutes.

  ```
  https://bridge.tempo.xyz
  From:  Ethereum, Arbitrum, or Base
  To:    Tempo Chain (Chain ID: 4217)
  Token: USDC to USDC.e
  ```

  **relay.link (Alternative)**

  Go to relay.link. Same process. Usually faster from Arbitrum.

  ```
  https://relay.link
  From:  Ethereum or Arbitrum
  To:    Tempo Chain (Chain ID: 4217)
  Token: USDC to USDC.e
  ```

  ## Step 3: Add Tempo Chain to Your Wallet

  | | |
  |---|---|
  | Network Name | Tempo Chain |
  | RPC URL | https://rpc.tempo.xyz |
  | Chain ID | 4217 |
  | Currency Symbol | USD |
  | Block Explorer | https://explore.tempo.xyz |

  In MetaMask: network dropdown, Add Network, Add a network manually, enter the settings above.

  ## Step 4: Add USDC.e as a Custom Token

  After bridging, add USDC.e to your wallet so you can see the balance. In MetaMask: Tokens, Import tokens, paste this address:

  ```
  0x20c000000000000000000000b9537d11c60e8b50
  ```

  ## Step 5: Buy $TROUTE

  Go to www.trouteprotocol.xyz. Connect your wallet. Enter the amount. Confirm. $TROUTE lands in your wallet in the same transaction. No claiming, no waiting.

  ## Troubleshooting

  **Cannot see USDC.e after bridging.**
  Check that your wallet is on Tempo Chain (Chain ID 4217) and that you have added USDC.e as a custom token using the address above.

  **Bridge is slow.**
  Ethereum mainnet bridges can take up to 15 minutes. Arbitrum is faster. If it has been over 30 minutes, check the transaction on your source chain explorer.
